# Dotfiles

Configuracion de terminal `alacritty`.

# Instalacion

    pacman -S alacritty
    yay -S nerd-fonts-hack

# Configuracion

## Nombre de las ventanas
Alacritty

## Dimensiones
120 columnas x 28 lineas

## Fuente
Nerd Font Mono 9

## Cursor

Estilo: Doble T 

Parpadeo: Cada 750ms si tiene el foco

Oculto mientras se escribe
